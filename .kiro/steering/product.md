# Product Overview

GeoCities Reborn is a retro website builder that combines 90s GeoCities nostalgia with modern AI-powered development tools.

## Core Features

- **AI-Powered Code Editor**: Monaco editor (VS Code engine) with Google Gemini AI assistant for code generation
- **Multi-File Projects**: Full file management (create, rename, delete, duplicate) with live preview
- **Themed Neighborhoods**: Publish sites to 12 GeoCities-style neighborhoods (Area 51, Hollywood, Silicon Valley, Tokyo, etc.)
- **Gallery & Discovery**: Browse and explore community-created retro websites
- **Profile Management**: Edit and manage published sites with view tracking
- **Retro Templates**: Blank, Cyber (Matrix), and Glitter themes with pre-configured files

## User Experience

- Retro 90s aesthetic (starry backgrounds, rainbow text, marquee, visitor counter)
- Professional development tools (syntax highlighting, IntelliSense, auto-completion)
- Real-time live preview with auto-save
- One-click publish with shareable links
- Download as single HTML file

## Target Use Cases

- Learning HTML/CSS/JavaScript
- Building nostalgic retro websites
- Quick prototyping with AI assistance
- Educational web development projects
